7. Como crear un proyecto

Desde la pantalla principal se pulsa el botón "+":

Se selecciona el tipo de presentación, por ejemplo “Presentation”. A continuación, se presentan una serie de plantillas predefinidas. Las marcadas con el logo “PRO” significa que solo están accesibles en la versión de pago.  Seleccionamos una plantilla que no tenga este logo, por ejemplo “5 Fun Tips”.

A continuación, pulsar en "Edit in Studio" para cargar el proyecto en el editor de diseño:

Se crea un proyecto proyecto basado en la plantilla seleccionada. Este proyecto contendrá las diapositivas con el contenido que tiene la plantilla, por lo que a continuación podemos ir personalizando las diapositivas ya existentes para introducir el contenido.
